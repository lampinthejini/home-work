    - `.env`를 git에서 제거하고 `.env.example` 템플릿으로 전환

## 실행 방법

### 1. 실행 전 준비
- `pip install -r requirements.txt` 로 필요한 패키지를 설치합니다.
- `.env.example` 파일을 복사해서 `.env` 파일을 만듭니다. (`cp .env.example .env`)
- Docker Desktop이 설치되어 실행 중이어야 합니다. (컨테이너 자체는 아래 2번에서 자동으로 만들어집니다)

### 2. run_desktop.py로 실행하기 (추천)
```
python run_desktop.py
```
- `hw02-postgres` 도커 컨테이너가 없으면 자동으로 만들고, 꺼져 있으면 자동으로 켭니다.
- 데이터베이스 마이그레이션(Alembic)을 자동으로 최신 상태까지 적용합니다.
- 데스크톱 창이 뜨면서 바로 서비스를 사용할 수 있습니다.

### 3. 서버만 직접 열어서 시작하기
데스크톱 창 없이 웹 서버만 쓰고 싶을 때 사용합니다.

1. 도커 컨테이너를 먼저 준비합니다. (`run_desktop.py`를 한 번도 실행한 적이 없다면 아래 명령으로 직접 만들어주세요. 이미 만들어져 있다면 건너뛰어도 됩니다.)
   ```
   docker run -d --name hw02-postgres -e POSTGRES_USER=user -e POSTGRES_PASSWORD=password -e POSTGRES_DB=contact_db -p 5432:5432 postgres:16
   ```
2. 서버를 실행합니다.
   ```
   uvicorn contact_app.main:app --reload
   ```
3. 브라우저에서 http://127.0.0.1:8000 으로 접속합니다.
