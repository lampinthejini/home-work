📒 연락처 관리 프로그램 (Python)
📖 프로젝트 소개

Python을 활용하여 구현한 콘솔(Console) 기반 연락처 관리 프로그램입니다.

회원 정보를 추가, 조회, 수정, 삭제(CRUD)할 수 있으며, 프로그램 종료 시 데이터를 파일에 저장하고 실행 시 다시 불러오는 기능을 제공합니다.

🎯 과제 목적

본 프로젝트는 Python 기초 문법과 자료구조를 활용하여 실제 프로그램을 설계하고 구현하는 것을 목표로 합니다.

특히 다음 내용을 중점적으로 학습하였습니다.

함수(Function)를 활용한 프로그램 구조화
리스트(List)와 딕셔너리(Dictionary)를 이용한 데이터 관리
예외 처리(Exception Handling)
정규표현식(Regex)을 활용한 입력값 검증
파일 입출력(Pickle)을 통한 데이터 영속성 구현

구현을 시작하기 전에 다음 문서를 먼저 작성한 후 개발을 진행하였습니다.

본 프로젝트의 설계 및 구현 과정에서는 AI를 사용하지 않았습니다.

요구사항 분석, 설계, 코드 작성 및 구현은 모두 직접 수행하였으며,
GitHub README 작성 등 프로젝트 문서 정리에만 AI의 도움을 활용하였습니다.
