# PostgreSQL 학습 과제 (1.5)

## 📖 프로젝트 소개

Docker 환경에서 PostgreSQL 16 데이터베이스를 구축하고, DBeaver를 이용하여 데이터베이스를 관리하는 실습 프로젝트입니다.

초기에는 CRUD(Create, Read, Update, Delete) 기능을 SQL로 구현하는 방법을 학습하였으며, 이후에는 다양한 SELECT 문법과 집계 함수, JOIN, 서브쿼리, 윈도우 함수 등을 활용하여 데이터를 조회하고 분석하는 SQL 작성 방법을 연습하였습니다.

---

## 🎯 과제 목적

본 과제는 관계형 데이터베이스의 기본 개념을 이해하고, PostgreSQL을 활용한 SQL 작성 능력을 향상시키는 것을 목표로 합니다.

주요 학습 내용은 다음과 같습니다.

* Docker를 이용한 PostgreSQL 16 실행
* DBeaver를 이용한 데이터베이스 관리
* SQL CRUD(Create, Read, Update, Delete) 작성
* SELECT 문을 활용한 데이터 조회
* JOIN을 통한 테이블 연계
* GROUP BY, HAVING을 활용한 집계
* 서브쿼리 및 CTE(Common Table Expression)
* Window Function을 활용한 데이터 분석

## 🤖 AI 활용 여부

본 과제는 **스스로 해결하는 것을 원칙**으로 진행하였습니다.

각 문제는 먼저 요구사항을 분석하고, 자료를 참고하며 **AI의 도움 없이 직접 끝까지 구현 및 완성**한 후에 검토를 진행하였습니다.

이후 작성한 SQL을 AI와 비교하여 다른 풀이 방식이나 개선점을 확인하였고, 마지막으로 **강사님의 피드백과 정답 예제(답지)**를 함께 비교하며 최종적으로 내용을 검증하였습니다.

즉, AI를 정답을 대신 작성하는 도구로 사용한 것이 아니라, **직접 완성한 결과를 검토하고 더 나은 작성 방법을 학습하기 위한 보조 도구**로 활용하였습니다.
